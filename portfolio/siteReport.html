Module Experience

Embarking on the journey of learning web development, particularly focusing on HTML and CSS, is a foundational step for anyone looking to enter the field of web design and development. This report describes my personal experience learning these essential technologies, highlighting the challenges faced, the strategies employed to overcome them, and the rewarding outcomes achieved.

1. Starting with HTML:

i. Basic Structure:My learning began with understanding the basic structure of HTML documents, including the use of tags, elements, and attributes. The simplicity of creating a webpage with just a few lines of code was both surprising and motivating.

ii. Content Organization:I learned how to organize content using various HTML tags such as headings, paragraphs, and links. This helped me grasp the semantic nature of HTML and its role in defining the structure of web content.

2. Diving into CSS:

i. Styling Basics:Transitioning to CSS, I was introduced to styling HTML elements. I learned about selectors, properties, and values, which form the core of CSS. The ability to change colors, fonts, and layouts was exciting and opened up creative possibilities.

ii. Box Model:Understanding the box model was crucial. It explained how elements are displayed on the web, including margins, borders, padding, and content. This knowledge was essential for precise control over layouts.

Learning and Resources

Platforms like W3school, Geeks for Geeks, and Javat points provided structured learning paths and interactive exercises that were instrumental in building my foundational knowledge.
