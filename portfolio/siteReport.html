Experience of learning web development

Embarking on the journey of web development has been both challenging and rewarding. This experience has allowed me to dive deep into
various aspects of creating a functional and visually appealing website.

Coding

Learning to code was the foundation of this project. My journey with HTML & CSS started with understaning the basic syntax and strucutured of these language.

HTML:

I learned to structure web pages using HTML elements. This involved creating headings, paragraphs, list, links, images and forms. I aslo learned how to use
semantic elements header, nav, iframe and footer and others to improve the readability and accessibility of the HTML code

CSS:

I learnde to style the HTML elemnts to make the web page visually appealing. This includes selectors and properties, Box model and Layout Technique.
Selectors and Properties is about how to target HTML elemets with selectors and apply various properties to style them where as Box model is about margin,
padding, border. And Layout Rechnique use features like flexbox, Grid to create resposive and flexible layouts that adapt to different screen size
